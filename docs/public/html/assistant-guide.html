AI 助手使用指南

3 分钟掌握正确的打开方式，让 AI 真正帮你提效
📢
这份指南已经讲过很多次了，也在会上当面演示过。请认真看完，有问题再问。
1
私聊直接说，群里要 @

✅ 私聊 直接发消息即可，不需要 @ 机器人

✅ 群聊 必须 @机器人 才会触发回复

为什么群里要 @？因为群里有多个人在说话，不 @ 的话机器人无法区分哪些消息是给它的，上下文会混乱。

2
先定义角色，再输入主题

AI 不是聊天工具，是生产力工具。正确的使用方式是：先校准身份，再批量生产。

第一步，告诉 AI 你是什么角色。它会生成一个专属工具模板。之后每次使用，只需要输入本次主题就够了。

🎬

口播脚本、选题策划
人设打磨、内容提案
📷

场景设计、视觉参考
✍️
编辑 / 运营
文案撰写、标题优化
数据解读、发布策略

✅ 高效用法

"我是摄影师，对标影视飓风，帮我生成一个用于脚本分镜拆解的Agent"
💡 一次定义，反复使用
定义一次："我是美食达人，风格是接地气的东北幽默，帮我生成一个口播脚本工具"
之后每次："主题：锅包肉" / "主题：烧烤季" / "主题：年夜饭"
——不用每次重复说你是谁、什么风格，AI 已经记住了。
3
创建 Agent 后，让它拉群

当你跟机器人私聊，让它创建了一个专属的智能体（Agent）后——

✅ 正确做法 让机器人自己创建群聊，它会自动配置好上下文，记住之前的对话

❌ 错误做法 自己手动拉群 → 新群里的机器人不记得刚才聊了什么，等于从零开始

正确的操作流程
① 私聊机器人，描述你的需求
② 机器人帮你创建 Agent
③ 告诉机器人："帮我创建一个群聊"
④ 机器人拉群，上下文无缝衔接 ✨
4
创建群聊时请耐心等待

机器人创建群聊的时候，后台需要：

📡 调用飞书 API 创建群
⚙️ 写入配置文件、绑定 Agent
🔄 重启服务使配置生效

整个过程可能需要 30 秒到 1 分钟，这是正常的，耐心等一下就好。

5
看到乱码别慌

如果收到了一大串英文字符或报错信息，说明服务暂时出了点问题。

✅ 正确应对 等 1-2 分钟，再重新发一次消息

❌ 不要做 疯狂重发、截图问人、觉得坏了就不用了

服务器偶尔抽风是正常的，通常很快就会自动恢复。如果超过 5 分钟还不行，联系管理员。
